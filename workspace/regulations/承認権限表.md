# 承認権限表

制定日: 2024年4月1日
最終改定日: 2025年10月1日

---

## 1. 購買承認権限

| 金額区分 | 承認権限者 | 備考 |
|---------|-----------|------|
| 50万円未満 | 購買部担当者 | 担当者判断で発注可 |
| 50万円以上100万円未満 | 購買部課長 | |
| 100万円以上500万円未満 | 購買部長 | |
| 500万円以上 | 取締役 | |

## 2. 代理承認

購買部課長が不在の場合は、以下の条件を全て満たす場合に限り、購買部担当者が代理承認を行うことができる。

1. 課長が出張等により2営業日以上不在であること
2. 金額が100万円未満であること
3. 事後に課長の追認を得ること

代理承認を行った場合は、代理承認の理由および課長の追認結果を記録するものとする。

## 3. 合算に関する規定

同一取引先に対する同一月内の発注は、承認権限の判定において合算するものとする。

ただし、発注の目的・用途が明確に異なる場合はこの限りではない。

合算の対象となるかの判断に迷う場合は、上位の承認権限者に確認することが望ましい。

## 4. 備考

- 上記金額は全て税抜金額とする。
- 緊急の場合の取扱いは購買管理規程第10条による。
- 承認権限者の不在時の取扱いは上記第2項による。ただし、購買部長以上の不在時の取扱いは個別に判断するものとする。
